VOWEL

Vowel
This MUST follow Short Word. Will cycle through vowels within Short Word.

VOWEL2

Vowel in position 2
YES: keep only words whose second letter is a vowel (A, E, I, O, U). NO: remove all words with a vowel in the second position. SKIP: leave the wordlist unchanged.

CONNECTIVE

CONNECTIVE
Enter a word. The app extracts unique consonants in order and keeps only words that start with one of those consonants. Example: "sunglasses" gives S, N, G, L. In Settings, toggle whether Y is treated as a vowel.

VOWEL POS

Vowel Position
This MUST follow Short Word, 3 LETTER WORD, or CONS MID. Will cycle through vowels within Short Word. Select BEGIN, MID, or END to specify which section of the word MUST contain the vowel.

NAME

NAME
Enter a full name (first and surname). The feature extracts the initials and uses them as consonants (like Short Word) to filter words. Optionally select a vowel and position (START/MID/END) to further filter words containing that vowel in the specified position; if you don't need it, just click SUBMIT after entering the name.

O?

O?
Filters whether or not there is an "O" in the chosen word.

O-CURVES

Two-part filter.
Part 1: Is there an O in the word? (YES/NO). Part 2: Enter which positions (1–5) have curved letters, e.g. 123 or 0 for all straight. Same logic as O? then Curved Pos.

Curved Pos.

Curved Positions
In which positions are there curved letters (as caps)- example. 135

ZERO CURVES

Curved positions 1–6.
Enter positions 1–6 where the letters are curved (e.g. 135, or 0 for no curved letters in 1–6). APPROX (Settings) relaxes the assumption that other positions are straight.

EEE?

E?
Input an "E", an "E" sound, or no E in the second position of the chosen word.

EEEFIRST

E?
Input an "E", an "E" sound, or no E in the first position of the chosen word.

E21

Three-part E filter.
Part 1: EEE? (second letter) – E, E-sound, or No E. Part 2: EEEFIRST (first letter) – again E, E-sound, or No E. Part 3 (optional, if enabled in Settings): ask if there is an E anywhere in the word (YES/NO). All three parts filter the same wordlist step.

ADV-LEX

Advanced Lexicon helper.
Uses the dynamic Lexicon position and its possible letters. For each letter, ADV-LEX picks one long example word from the 19127 list that contains that letter but none of the other Lexicon letters. You choose which word matches your puzzle, and the wordlist is filtered to that letter at the chosen position. SKIP leaves the list unchanged.

Lexicon

Lexicon
Note the lexicon position and possible letters. Enter a word beginning with the letter in that position within the chosen word.

Cons. Together

Consonants Together?
Are there two consonants together within the chosen word?

COLOUR3

Colour in 3rd position.
Can the spectator think of a colour beginning with the letter in the 3rd position? Yes/No.

ATLAS

Colour letters in positions 1–6.
Enter how many of the first 6 letters can start a colour name (0–6). Words are filtered to those that match that count. Same letter set as COLOUR3 (e.g. A, B, C, E, G, I, L, N, M, O, P, R, S, T, V, W, Y).

Letter Lying

Multi-part filter: is this letter in your word?
Configure in Settings (Static: fixed letter string, or Dynamic: most frequent letter each step). For each letter you get YES / NO (filter by presence) or skip that letter (→) or SKIP the whole filter.

Love Letters

Number string from letter groups.
Spectator writes letter groups on a card (default: ABC DEF GHI JKL MNO), then for each row writes how many distinct letters from that row appear in the word. You enter that number string (e.g. 11112). Configure groups in Settings.

LENGTH

Word Length
Type in a number referring to the length of the word.

LETTER SHAPES

Position + shape
Choose a position (1–6, counting from the first letter), then STRAIGHT, MIXED, or CURVED. Words shorter than that position are removed. The letter at that position must belong to the chosen set:
STRAIGHT: A, E, F, H, I, K, L, M, N, T, V, W, X, Y, Z
MIXED: B, D, G, J, P, Q, R, U
CURVED: C, O, S, G, Q, U

MOST FREQUENT

Most Frequent Letter
The most frequent letter in the combined wordlist. (Can be used more than once)

LEAST FREQUENT

Least Frequent Letter
The least frequent letter in the combined wordlist. (Can be used more than once)

ABSENT

Letters NOT in the word.
Type in a string of letters. Any word containing at least one of these letters will be removed from the wordlist.

PRESENT

Letters that ARE in the word.
Type in a string of letters. Only words that contain all of these letters (in any positions) will be kept.

CONSONANTS

Two consonants filter
Enter exactly two consonants (e.g. RP), then choose: TOGETHER (both letters adjacent, RP or PR), MIDDLE (both in the word, neither first nor last letter), or ANYWHERE (both in the word, any position).

ANYWHERE

Anywhere
Enter exactly two consonants (e.g. RP). Only words that contain both letters are kept; position and order do not matter.

TOGETHER

Together
Enter exactly two consonants (e.g. RP). Only words where those two letters appear adjacent (RP or PR) are kept.

MIDDLE

Middle
Enter exactly two consonants (e.g. RP). Only words that contain both letters and neither letter is the first or last character are kept.

ABCDE

Are there the letters A-E within the word? Answer YES for individual letters, then DONE to confirm.

ABC

Are there the letters A-C within the word? Answer YES for individual letters, then DONE to confirm.

FIND-EEE

Select a letter that is definitely present in the target word. Words will be filtered to only include those containing the selected letter. Only one letter can be selected at a time.

POSITION-CONS

POSITION-CONS
Filters words by a specific position (leave blank to check all positions) that must contain one of the letters from a provided string. Enter the total number of letters from that string (including the position letter) that appear in the target word.

FIRST CURVED

FIRST CURVED
Filters words where the first curved letter appears at a specific position. All letters before that position must be straight letters, and the letter at the specified position must be curved.

EYE TEST

EYE TEST
Automatically builds 6 PIN-style letter groups (sizes 2, 3, 4, 5, 6, and 7) from the current wordlist, chosen for high information gain. Each box is pre-filled with its group; you enter a 6-digit code where each digit is how many letters from that group appear in the target word. It uses the same matching logic as PIN but with dynamic groups, acting like an \"eye test\" for the wordlist.

NEW

NEW
Shows four letters per batch: the next most and least frequent consonants and vowels (A,E,I,O,U only — Y is not a vowel here). NEW advances to the next batch. After 10 seconds on a batch, or when you press ANSWER, that batch is committed as a stop: first stop = first letter; second = second letter; further stops = letters anywhere in the word. Enter digits (e.g. 112) for how many extra letters from each batch (see in-app help). MOVE ON finishes the step. Optional +1 buffer in Settings allows each answered count to match the digit you entered or one higher (not one lower).

PIN

PIN
Enter up to 6 words and a corresponding code. Each digit in the code represents how many letters from the corresponding word appear in the target word. The feature filters words that match all code requirements.

SCRABBLE

SCRABBLE
Enter a number representing the total Scrabble score of the word. The filter will keep only words whose letter values sum to the entered score. Letter values: A,E,I,O,U,L,N,S,T,R = 1; D,G = 2; B,C,M,P = 3; F,H,V,W,Y = 4; K = 5; J,X = 8; Q,Z = 10.

SCRABBLE1

SCRABBLE1
Enter a score. The filter keeps words whose Scrabble score is within ±1 of that number (e.g. 8 keeps 7, 8, and 9). Words that match the exact score you entered are highlighted in blue. Same letter values as SCRABBLE.

SOLOGRAM

SOLOGRAM
Uses a secondary wordlist (pointed-to words). Enter a string of Y and N: one character per letter of the word the spectator is pointing at. Y = that letter is in their thought word; N = that letter is not. The string length gives the pointed-to word length. Words where the same letter has both Y and N are excluded. The main list is filtered to words consistent with at least one remaining pointed-to word.

DECODE

DECODE
Enter a string of letters. Word length must match. One character in the string is correct at its position in the word; the rest are wrong. By default (Settings Position OFF) any one of them can be the correct one. In Settings you can turn Position ON and set which position (1-based) is the correct one.

PIANO FORTE

Piano Forte
Select letters A-G that MUST be in the word. Selected letters must be present, and unselected letters must NOT be present. Click SUBMIT to filter, or NONE if none of the letters A-G are in the word.

PIANO PIANO

Piano Piano
Select a number (1-7) representing the total count of letters A-G in the word. The word must contain exactly that many A-G letters in any combination.

LENGTH

T9 Length
Type in a number referring to the length of the T9 number string for the word. The word's T9 representation must match this length exactly.

LAST TWO

T9 Last Two
Select 2 digits (2-9) to check if the T9 string ends with those digits. Only words whose T9 representation ends with the selected digits will be kept.

LAST

T9 LAST
GUESS (optional) and ACTUAL (required). Enter ACTUAL as 1 or 2 digits (2-9). Words are filtered so the last 1 or 2 digits of their T9 string match ACTUAL. If GUESS is empty, only ACTUAL is used.

POSITION 5

T9 POSITION 5
GUESS (optional) and ACTUAL (required). Enter ACTUAL as 1 digit (2-9). Words are filtered so the digit at position 5 of their T9 string matches ACTUAL. SEND TO HYDRA uses GUESS + ACTUAL sum. GUESS filter works after POSITION 5.

GUESS

T9 GUESS
Only works after LAST or POSITION 5 in a workflow when a GUESS value was entered. For each GUESS digit, you answer YES or NO: “Is this digit in the first 4 digits of your T9 string?” Words are filtered accordingly.

1 LIE (L4)

T9 1 Lie (L4)
Select 4 digits (2-9) representing the last 4 digits of the T9 string. The filter assumes exactly ONE of these digits is incorrect. Words will be kept if their T9 string's last 4 digits match any scenario where 3 digits are correct and 1 digit is different.

Repeat?

T9 Repeated Digits
Filters words based on whether their T9 string contains two consecutive repeated digits. Select YES if the T9 string has repeated digits next to each other, or NO if it does not.

Higher?

T9 Second Digit Comparison
Filters words based on the relationship between the first two digits of the T9 string. Select YES if the second digit is higher than the first, NO if it's lower, or SAME if both digits are equal.

NUMBER START

First T9 digit range
Like Dictionary (Alpha) but for the first T9 digit. Choose Low (2–4), Mid (4–6), or High (6–9). Use before SINGING to narrow by first-digit range, then apply directions for exact length.

1 TRUTH (F4)

T9 1 Truth (F4)
Select 4 digits (2-9) representing the first 4 digits of the T9 string. The filter assumes exactly ONE of these digits is correct. Words will be kept if their T9 string's first 4 digits match any scenario where 1 digit is correct and 3 digits are different.

AlphaNumeric

AlphaNumeric Rule
YES: Keeps words where the word length matches a letter's position in the alphabet AND that letter is in the word (e.g., 1-letter words with A, 2-letter words with B, etc.). NO: Removes words that follow the rule, keeping all others.

Letters Above

Letters Above
Enter a number (0-26) representing how many letters before a secret chosen letter must be in the word. Words are kept if there exists at least one letter in the word such that exactly that many letters before it (in the alphabet) are also present in the word.

Dictionary (Alpha)

First Letter Location
Filters words based on the first letter's position in the alphabet. Default bands: Beginning A–M, Middle I–T, End N–Z. Custom Beginning/Middle/End letter ranges are set under Settings → ALPHA (First-letter bands). Select a section and click SUBMIT.

ALPHA (SHORT)

Directional alphabet filter (short)
The spectator imagines the alphabet in front of them and mentally spells the word by pointing to each letter. You enter Left (toward A), Right (toward Z), or Repeat (same letter as previous). If Dictionary (Alpha) ran earlier, the first letter is limited to that Beginning/Middle/End band (letter ranges for those bands are under Settings → ALPHA); if not, the first letter may be any A–Z. No on-screen B/M/E or doubles options—use the full ALPHA step for that. Set “Number of directions” in Settings to auto-submit after N entries, or 0 to submit manually.

ALPHA

Full directional filter
Optional: choose Beginning/Middle/End for the first letter (leave unset for any A–Z), optional YES/NO for consecutive identical letters in the spelling (leave unset to ignore), optional possible last letters, then enter your Left / Right / Repeat sequence and SUBMIT once. What each band contains is set under Settings → ALPHA → First-letter bands. Uses the same direction-count and Swap POV as ALPHA (SHORT).

REPEAT?

Repeated letters in the spelling
Filters words based on whether their spelling contains at least one instance of consecutive identical letters (AA, LL, …). Select YES if there is a consecutive identical pair, or NO if there is not.

ALPHA-WORD

ALPHA + possible last letters
Same directional spelling as ALPHA. Word length must be exactly one more than the number of directions. Enter possible last letters in the box (letters only; duplicates are ignored, e.g. SHEEP → S, H, E, P). Only words whose final letter is in that set are kept. First-letter band matches Dictionary (Alpha) if that step ran earlier; otherwise any A–Z. Uses the same Settings as ALPHA (direction count, Swap POV).

ALPHA-REPEAT

Repeated letters in the spelling
First choose YES or NO for consecutive identical letters in the word. NO: words with any double letter (AA, LL, …) are removed; directions apply letter-by-letter; length = 1 + directions. YES: only words with at least one such double; consecutive runs collapse to one letter for the directional path (e.g. BUSSES → BUSES for the path), and collapsed length must be 1 + directions. Same first-letter rules and Settings as ALPHA (Dictionary band if that step ran, else any A–Z).

S/M/L (Length)

Word Length Categories
Filters words by length: Small (1-6 characters), Medium (5-9 characters), Long (7+ characters). Select a category and click SUBMIT.

OMEGA

Positional shape filter
Tap shapes in order to build a sequence (each shape has a set of allowed letters). Only words that match the sequence at every position are kept. Use Backspace to remove the last shape, then SUBMIT when done. Mode (ESP, Pocket, Restaurant) is set in Settings.

CALCULUS

Positional digit filter
Like OMEGA Short but with digits 0–9. Each digit maps to a fixed set of letters. Tap digits to build a sequence; only words with length ≥ (sequence length + 1) whose letters at each position match that digit's set are kept. Same forbidden pairs as OMEGA. Leading zero allowed. Use Backspace, then SUBMIT.

THE CORE

Consonants + helper word + vowels.
First, you ask if there are two consonants together in the thought-of word. If YES, the spectator gives a short helper word; any two of its consonants must appear together somewhere in the word. If NO, they give a helper word whose consonants may all appear in the word, but never together. Finally, you run through the distinct vowels from that helper word one by one, confirming whether each vowel is in the word and, for each vowel, where in the word it lands.

NUMEROLOGY

Find months/dates from a numerology number.
Enter a numerology number (1–9) and a difference. The filter finds dates in the year whose day/month digits reduce to that number with the given difference and keeps only words whose MONTH matches those dates in the MONTHS_STARSIGNS list (shared BIRTHDAY results update under the feature).

BINARY

Binary word-filter tools.
Opens the BINARY tools (MUTE and MUTE DUO). These filters split the word list using YES/NO style answers to letters of the alphabet, showing two possible interpretations in parallel. Use the BINARY button to access the individual tools.

MUTE

Single-spectator binary filter.
At each step you see a letter; the spectator answers LEFT or RIGHT. The app keeps both logical interpretations in play (LEFT = “letter is in” vs RIGHT = “letter is out”, and vice-versa) and updates the LEFT/RIGHT word lists and counts after each answer.

MUTE DUO

Two-spectator version of MUTE.
Use the D-pad style buttons (1&4, 2&4, 2&3, 1&3) so a single press applies a paired YES/NO decision to both spectators at once. The four SPEC boxes show the two interpretations for each spectator side by side, updating after every answer.

T9

T9 Features Category
Click this button to view and access T9-based filtering features. T9 features filter words based on their phone keypad number representation (A-Z mapped to 2-9).

Alpha-Numeric

Alpha-Numeric Features Category
Click this button to view and access Alpha-Numeric filtering features. These features filter words based on relationships between letters and their positions in the alphabet.

Workflow REPORT

Use COPY on each run’s header to copy a readable summary plus JSON (separated by a line of dashes). Expand a run to set the debug target word (type A–Z, then Save), view elimination trace, and browse steps; expand a step for duration, traces, and payload. Steps that leave 0 words are highlighted in red.
